IT General Guide Solutions for daily problems

Windows Image Setup

  • Use Rufus or Windows Media Creation Tool
  • USB drive at least 8 GB
  • Choose appropriate ISO (Windows 10/11 Pro)
  • GPT for UEFI - MBR for Legacy

  • Boot from USB (F12, ESC, Del)
  • Choose Custom: Install Windows only
  • Delete all Partitions
  • Select Unallocated Space and press Next

  • Check Drivers
  • Run Windows Update
  • Install apps: Office, Company Portal, AnyDesk
  • Join Domain if required
Performance Issues
  • RAM: If 90%+, needs upgrade
  • HDD: Upgrade to SSD is fastest fix
  • CPU: Check what's consuming processor
  • Storage: Clear %temp% and empty recycle bin
  • Restart: Fixes 80% of problems
Computer Not Working
🔌 Power Issues:
  • Check cable
  • Try another socket
  • Remove battery 30 seconds
  • Try different power adapter
🖥️ Boot Issues:
  • Try external monitor
  • Listen for beep sounds
  • Try Safe Mode
  • Clean RAM modules
Headset Issues
  1. Check if not muted
  2. Verify it's visible in Playback devices
  3. Try different USB port
  4. Test on your phone
  5. Update/Rollback driver
