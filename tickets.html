Tickets Guide Data required before opening a ticket

Access Request (App / VPN)

Any request for access to a new application or VPN privilege.

Required Data:
  • Line Manager Username
  • User Mobile Number
  • Application Link (URL)
  • Business Need
  • Attach Screening Error (if available)

📋 Ready Template:

"Access Request for [Application Name] - User: [Name] - Business Need: [Reason] - Link: [URL]"

Avaya / Communication Issue

Voice system issues, call problems.

Required Data:
  • Line Manager Username
  • User Mobile Number
  • Extension Number or Avaya ID
  • Attached Escalation Form
  • Detailed Problem Description

📋 Ready Template:

"Avaya Issue for User [Name] - Extension [Number] - Problem: [Detailed Description]"

Port / Network Issue

Internet not working, port is down, cable issue.

Required Data:
  • Line Manager Username
  • User Mobile Number
  • Port Number
  • Seat Number
  • Device MAC Address

📋 Ready Template:

"Network Issue for User [Name] - Seat [Number] - Port [Number] - MAC: [Address]"

Cisco Issue

Cisco headset issues and communications.

Required Data:
  • Line Manager Username
  • User Mobile Number
  • Headset Model
  • Exact Problem Description
  • Steps Already Taken

📋 Ready Template:

"Cisco Issue for User [Name] - Model [Number] - [Problem Description]"
